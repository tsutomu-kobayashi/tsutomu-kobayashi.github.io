# Rodin API Documentation (Republished)

This repository republishes generated API documentation based on the Rodin Platform (https://sourceforge.net/projects/rodin-b-sharp/).

## Overview

* This repository contains HTML documentation generated using Javadoc.
* The content (e.g., class names, method signatures, and comments) is derived from the Rodin Platform.
* No modifications have been made to the original documentation content.
